# 💼 Personal Portfolio Website

A modern and responsive **Portfolio Website** built using **HTML, CSS, and JavaScript** to showcase projects, skills, and contact information in a professional way.

---

## 🚀 Features

* 👤 About Me section
* 🛠️ Skills showcase
* 📂 Projects section
* 📞 Contact section
* 📱 Fully responsive design
* 🎨 Clean and modern UI

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript

---

## 📂 Project Structure

Portfolio-Website/
│── Portfolio.html
│── app.css
│── script.js
│── images (png, jpg, svg files)

---

## ▶️ How to Run

1. Clone the repository
   git clone https://github.com/siddharth182005-dotcom/Portfolio-Website.git

2. Open the folder

3. Open `Portfolio.html` in your browser
